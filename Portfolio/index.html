Paladins Stats

A program that runs as a service on my computer that gathers match data from the Paladins API, storing all matches played in a PostgreSQL database. Uses SQL queries to generate winrates for different champions/talents at various ranks and export them to csv files to be used in spreadsheet software for further analysis.

 

Roguelands Mods

Minimap

A mod that draws a pixel art minimap of the current planet. By default works for vanilla, but also exposes an API that allows other terrain generating mods to interface with it.

ItemFrames

A mod that adds a custom Item Frame tile for your ship, allowing you to insert/remove items. Includes multiplayer support (items in item frames are visible to other players and update across the network) and saving the item frame and the held item's data to the save file.

Dresser

A mod that adds a custom Dresser tile for your ship, allowing you to change your character's augment and uniform at any time. Includes a ton of UI code to instantiate the custom menus and hook up all the buttons via code. Works with GadgetCore's extended Augment/Uniform registry.

TerrainGenerators

A mod that completely overhauls vanilla terrain generation with custom generators created for each planet. Each planet's textures are procedurally generated from 4 base textures in a way that allows the tiles to connect seamlessly in any orientation and configuration. Works in multiplayer. Put on hold after implementing 3 of 14 planet types due to the amount of effort it was to design, implement, and test unique terrain generators for each planet, as well as ensuring compatibility with other mods.
